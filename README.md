## ✨ Ключевые особенности

### 🛍️ Пользовательский опыт
- **Премиальный UI/UX:** Потрясающий дизайн, ориентированный на мобильные устройства, с плавной анимацией (AOS), эффектами glassmorphism и высокоадаптивной сеточной системой.

- **Динамический поиск товаров:** Динамическая навигация по категориям, интеллектуальный поиск и фильтрация в реальном времени.

- **Аутентификация:** Безопасный вход по электронной почте и паролю, а также интеграция с **Google OAuth** на базе Better Auth.

- **Панель управления пользователя:** Специальный портал для управления профилями пользователей, отслеживания истории заказов и загрузки счетов-фактур в формате PDF.

- **Корзина покупок и список желаний:** Постоянное управление состоянием для беспроблемного оформления заказа.

### 🛡️ Возможности администратора
- **Защищенная панель администратора:** Контроль доступа на основе ролей, защищающий панель управления.

- **Управление запасами:** Полное управление CRUD-операциями для товаров и динамических категорий.

- **Отслеживание заказов:** Видимость заказов клиентов, их статусов и выполнения в режиме реального времени.

- **Управление медиафайлами:** Автоматическая загрузка и оптимизация изображений с помощью Cloudinary.

## 🛠️ Технологический стек

### Фронтенд (клиент)
- **Фреймворк:** React.js (Vite)
- **Стилизация:** Tailwind CSS + Vanilla CSS (пользовательские полосы прокрутки, анимации)
- **Маршрутизация:** React Router DOM v6
- **Управление состоянием:** React Context API
- **Иконки и пользовательский интерфейс:** React Icons, AOS (анимация при прокрутке)
- **Утилиты:** jsPDF (генерация счетов)

### Бэкенд (сервер)
- **Среда выполнения:** Node.js
- **Фреймворк:** Express.js
- **База данных:** MongoDB с Mongoose ODM
- **Аутентификация:** Better-Auth (библиотека аутентификации нового поколения)
- **Хранилище:** Cloudinary (через Multer)
